## 环境配置(MacOS)
项目采用vue前端框架，前提需要安装node.js 及 vue
```
brew install node
npm install vue-cli -g
npm install vue
```
## 运行项目：
```
npm run build 编译项目在项目目录下生产dist静态文件，发布到nginx即可
如果是本机测试 运行测试环境：npm run dev
```
